#### 프로젝트 개요
시각장애인도 키오스크를 사용할 수 있게 기존 키오스크에 점자 변환기를 부착하여 사용하는 것을 목표로 라즈베리파이와 회로를 구성하여 진행 하였습니다. 키오스크 환경을 구성하기 위해 간단한 웹 프로그래밍을 통해 키오스크 페이지를 구성하였습니다.
